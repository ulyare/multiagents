Автор: Репичева Ульяна Дмитриевна. Прикреплены две фотографии, на одной тополгия графа и ручной проход алгоритма по этапам, на второй рассчёт сложности.

Описание алгоритма. На первом этапе агенты знают только своё число и свои связи с другими агентами. На втором этапе каждый агент передаёт всем доступным агнетам имеющиеся данные. На третьем этапе аналогичные действия. И так до тех пор пор, пока все агенты не узнают обо всех. После этого каждый агент вычисляет среднее арифметическое.

ИНСТРУКЦИЯ ПО ЗАПУСКУ!
1. В гитхабе нажать code и скопировать ссылку https.
2. Скачать jade из архива.
3. Открыть IDEA. File -> New -> Project from version control. Вставить ссылку и нажать clone.
4. Открыть File -> Project Structure -> Modules -> Dependencies. Нажать на плюсик и добавить JARs or directories. В окошке указать путь к jade.jar. Нажать ОК.
5. Можно запускать код
